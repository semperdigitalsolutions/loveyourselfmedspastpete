# Love Yourself Med Spa — 20% Off Webpage Kit

This download contains a buildable landing page starter and every major section from the selected mockup broken into reference slices.

## What is included

### Build files
- `index.html` — full static landing page markup
- `styles.css` — responsive styling based on the selected design
- `script.js` — placeholder for Google Ads conversion hooks
- `content.json` — structured copy and page content
- `design-spec.md` — colors, fonts, layout notes, and production checklist

### Assets
- `assets/logos/` — transparent logo icon, 1200x1200 logo, 600x600 logo, editable SVG lockup
- `assets/icons/` — line icons used by the page
- `assets/images/` — placeholder med spa imagery and service-card image crops

### Reference mockup slices
- `reference-mockup/full-page-reference.png`
- `reference-mockup/sections/` — header, hero, trust strip, services, why/testimonial, CTA, footer
- `reference-mockup/service-cards/` — each service card as a separate reference image

## Fastest way to use
1. Unzip the folder.
2. Open `index.html` in a browser.
3. Replace placeholder images with approved business photos.
4. Update the booking links to the exact booking URL.
5. Connect the form to your form provider or remove the form.
6. Add Google Ads conversion tracking for calls, bookings, and form submissions.
7. Upload to the website or deploy as a standalone landing page.
